Description
-----------
Fully configurable dice including shape (square or rounded), size, curve, resolution and the ability to embed modifiable characters in the face. Uses the 8 bit font module 'bitmap.scad' available from here: thingiverse.com/thing:2054

Instructions
------------
Simply change the parameters in OpenSCAD and generate the required stl file for rounded or traditional square-edged dice and embed any character in the face of the dice. Requires 'bitmap.scad' to be present in the same folder.

Note: if printing round-edge dice, some support may be required.
